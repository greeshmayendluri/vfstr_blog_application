# VFSTR Blog Application

## 🚀 Overview
VFSTR Blog Application is a role-aware, branch-segmented campus blogging platform designed for university students to create and manage blogs within the university ecosystem.

This repository currently contains the backend implementation of the application.

---

## ✨ Backend Features

- User authentication & authorization
- Role-based access control
- Blog CRUD operations
- Branch-wise content management
- Comment and interaction APIs
- Admin moderation functionalities
- RESTful API architecture

---

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone <repository-url>
```

### Navigate to Project Folder

```bash
cd vfstr-blog-application
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

### Run Server

```bash
npm start
```
